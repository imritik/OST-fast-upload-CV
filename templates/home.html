Documentation

There are following endpoints.
  1. cv-extract-api-2000.herokuapp.com/api/extract

    • Method allowed - POST
    • Paramaters - files[] - File/Files in pdf format.
    • Returns - Name, Email, PhoneNo, Experience, Original Text extracted from the CV in json format.
      • Example : 
                      
                    import requests

                    url = 'http://127.0.0.1:5000/api/extract'
                    files = {'files[]': open('data/sample.pdf', 'rb')}
                    
                    r = requests.post(url, files=files)
                    r.json()

  2. cv-extract-api-2000.herokuapp.com/api/addname

    • Method allowed - POST
    • Paramaters - name - first names or list of names to be added seperated by space. Ex - name1 name2 name3
    • Returns -{ message : success } if word is added, else { message : error }

  3. cv-extract-api-2000.herokuapp.com/api/keyword

    • Method allowed - POST / GET
    • POST - takes a .txt file with key "file[]" - text file with keywords seperated by comma.
    • GET - return current set of keywords.
